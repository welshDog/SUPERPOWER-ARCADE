# SUPERPOWER ARCADE

A strengths-first neurodivergent game that reveals hidden superpowers through playful challenges, ethics forks, and pattern-based scoring.

## What it does
- Runs a browser-based arcade quest.
- Tracks focus, creativity, ethics, teamwork, and problem-solving.
- Shows a fun result screen for players.
- Saves deeper profile data for private review.

## How to run
1. Open `index.html` in a browser.
2. Or serve it locally with your preferred static server.

## Project structure
- `index.html` app shell.
- `style.css` arcade styling.
- `app.js` game flow and scoring.
- `data/` challenge data.
- `docs/` design notes.
- `admin/` private dashboard.

## Goal
Find talented, good-hearted, neurodivergent thinkers who could be great teammates.
